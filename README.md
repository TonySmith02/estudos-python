## Introdução ao Python

A criação do python foi realizada por Guido van Rossum, Holandês e progrmador, a criação ocorreu no final dos anos 80 e início dos anos 90.

A primeira versão do python não foi a 1.0 mas sim a 0.9.0, sendo lançado em 1991. Seu nome foi escolhido como uma homenagem ao grupo de comédio britânico, Monty Python, do qual o Guido van Rossum foi um grande fã.

O objetivo da criação do python é para ser uam liguagem fácil de realizar a leitura mas também ser fácil de realizar a escrita do mesmo, assim tendo uma sintaxe clara e concisa.

Com essas características, a evolução do Python foi constante com o passar dos anos, assim evoluindo e ganhando popularidade e aos poucos se tornando uma das linguagens de programação mais utilizadas no mundo.

## Python e suas características
### Fácil Leitura
Python foi criado para se ter uma sintaxe clara e simples, o que faz com que a leitura e compreensão do código. Utilizando uma boa identação, delimitando os blocos de código, promove tanto uma boa estrutura e quanto uma boa leitura.

### Multiplataforma em Sistemas Operacionais
O Python tem a capacidade de ser executado em diferentes sistemas operacionais, como por exemplo o Windows da Microsoft, quanto o MacOS da Apple e também o Linux, sem a necessidade de realizar modificações no código. Assim o tornando uma linguagem versátil e portátil.

### Dinâmismo na Tipagem
Em Python, não é necessário realizar explicitamente o tipo de dados das variáveis. O Python deduzde maneira automática o tipo de dado com base no valor atribuído a uma variável, o que deixa simples a escrita de código.

### Ampla biblioteca padrão
A biblioteca padrão qu o Python fornece é extensa com uma grande quantidade de módulos e funções para realizar diversas tarefas, como por exemplo: Manipulação de arquivos, conexão a bancos de dados, processamento de texto  e entre outros.

### Interpretação
Python é uma linguagem interpretada, então, a execução do código é feita linha por linha em tempo real. Assim permitindo um ciclo de desenvolvimento rápido como também facilitando a depuração do código.

### Comunidade ativa
Graças ao contínuo crescimento da popularidade do Python, atualmente ele conta com uma comunidade de desenvolvedores grande e ativa, o que faz com que a contribuição com bibliotecas, frameworks e ferramentas adicionais seja constante e significativa. Isso mostra uma grande quantidade de recursos e suporte disponíveis.

## Aplicações do Python - A linguagem é utilizada em uma ampla gama de aplicações e domínios, como por exemplo:

### Desenvolvimento Web
Python é popularmente e utilizado para o desenvolvimento web backend, com o auxilio de frameworks populares, como por exemplo o Djongo e Flask.

### Ciência de dados
Python é a linguagem preferida para a realização de análise de dados, assim como a ciência de dados também, o motivo é as bibliotecas como a Numpy, Pandas e Matplotlib.

### Inteligência artificial e Machine Learning
Python é a principal escolha para projetos de IA e Machine Learning, graças a biblioteca TensorFlow e Scikit-learn como exemplos.

### Automatização de tarefas
Python pode ser utilizado para automatizar tarefas repetitivas, como processamento de arquivos, web scraping (Extração automatizada de dados de sites usando programas que navegam pelas páginas) e testes de software.

### Desenvolvimento de jogos
Python é amplamente utilizado no desenvolvimento de jogos simples, especialmente com as bibliotecas diponíveis, como por exemplo a Pygame.
